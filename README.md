# Coffee Delivery

<p> ainda não está responsivo, mas a idéia aqui foi demonstrar minhas habilidades com ReactJS </p>

link do projeto funcionando: https://coffedelivery.vercel.app/


![image](https://user-images.githubusercontent.com/77198706/196018916-024b0a39-41e2-4018-af45-7c4e41c77251.png)

<h1>Conceitos utilizados</h1>

<ul> 
<li> Estados </li>
<li> ContextAPI </li>
<li> LocalStorage </li>
<li> Imutabilidade do estado </li>
<li> Listas e chaves no ReactJS </li>
<li> Propriedades </li>
<li> Componentização </li>
</ul>

<h3>Neste projeto vamos encontrar as seguintes funcionalidades</h3>

<ul> 
<li> Listagem de produtos (cafés) disponíveis para compra </li>
<li> Adicionar uma quantidade específicas de itens no carrinho </li>
<li> Aumentar ou remover a quantidade de itens no carrinho </li>
<li> Formulário para o usuário preencher o seu endereço </li>
<li> Exibir o total de itens no carrinho no Header </li>
<li> Exibir o valor total da soma de itens no carrinho multiplicados pelo valor </li>
</ul>
